## 📌 Overview
This repository contains the core on-chain architecture for a Real World Asset (RWA) Vault using Zero-Knowledge Proofs (ZK-SNARKs). The smart contracts allow users to cryptographically prove their solvency or identity to the blockchain without ever revealing their underlying private data.

## 🏗️ Architecture
The system is divided into two main smart contracts:

1. **`Verifier.sol`**: An auto-generated cryptographic verifier created via SnarkJS and Circom. It contains the elliptic curve mathematics required to validate the Groth16 proof off-chain.
2. **`BovedaRWA.sol`**: The main business logic contract. It interfaces with the Verifier to authenticate users. If the ZK-proof is mathematically valid, the vault grants access or mints the RWA token; if manipulated, the transaction reverts instantly.

## ⚙️ Tech Stack
* **Circom & SnarkJS:** Used to compile the cryptographic circuits and generate the `.zkey` proving key.
* **Solidity:** Smart contract development.
* **Remix IDE:** Used for testing, compiling, and deploying to the Ethereum Virtual Machine (EVM).

## 🚀 How to Audit
1. Deploy `Verifier.sol` on any EVM-compatible network.
2. Deploy `BovedaRWA.sol` passing the Verifier's address in the constructor.
3. Call the `validarIdentidad` function injecting the parameters from `proof.json` and `public.json` generated by the off-chain auditor.

# ⚖️ Private RWA Solvency Protocol

### Overview
This protocol allows financial institutions to prove their solvency and asset backing (RWA) using **Zero-Knowledge Proofs (Groth16)**. 

### Key Features
- **Privacy-First:** Validates balances without exposing sensitive financial data on-chain.
- **ZK-SNARKs:** Utilizes the BN128 curve for high-security mathematical verification.
- **EVM Compatible:** Ready for deployment on Ethereum, Polygon, or private sidechains.

### Tech Stack
- **Smart Contracts:** Solidity 0.8.0 (Verifier & Vault)
- **Backend:** Python 3.12 (Web3.py, py-solc-x)
- **Frontend:** Streamlit (Stealth Aesthetic)
- **ZK-Circuits:** Circom & SnarkJS
